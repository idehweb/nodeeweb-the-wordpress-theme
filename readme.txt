== Description ==

Nodeeweb is wordpress theme which is trying to cover useful tools for free. We try to provide you standard tools for your website, that you won't need to install more plugins. we are trying to make your admin area tidy and clean.



== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in `Nodeeweb` in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Install' and then 'Activate' button to use your new theme right away.
4. Go to INSERT ABOUT PAGE for a guide on how to customize this theme.
5. Navigate to Appearance > edit in your admin panel and customize to your taste.
